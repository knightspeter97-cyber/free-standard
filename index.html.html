About

What this site is

A clean, fast, single-file editorial site. You can host it anywhere and edit the article/watch objects in the script to publish.

How to publish

  1. Edit the articles object in the script and paste HTML inside content.
  2. Edit the watches object with YouTube links + embed URLs + commentary.
  3. Add a new card in the Articles/Watch grids that calls openArticle('key') or openWatch('key').

Notes

  • This template is fully static—no backend required.
  • “Listen” uses the browser’s built-in text-to-speech.
